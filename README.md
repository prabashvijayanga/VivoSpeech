VivoSpeech – Text to Speech

VivoSpeech is a modern React-based Text-to-Speech web application that converts written text into natural, human-like speech using the VoiceRSS API.
It features a sleek black and gold design, built to be responsive and professional — ideal for showcasing front-end and API integration skills.

Features

Converts text to high-quality, natural-sounding speech

Multiple voice options (male/female, UK/US accents)

Stylish black and gold UI

Fully responsive for all devices

Option to download generated speech as an MP3 file

Built entirely with React

Tech Stack

React.js – Front-end library for building user interfaces

VoiceRSS API – Text-to-Speech audio generation

HTML5

CSS3 – Responsive and modern styling
